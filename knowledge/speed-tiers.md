# Speed Tiers

Purpose: decide whether blitz arguments are valid.

Source reference: https://vsbattles.fandom.com/wiki/Speed

## Practical logic

- Comparable speed: both sides can normally interact.
- One rank gap: advantage, not automatic blitz.
- Two or more rank gaps: blitz is plausible if the faster side has range, awareness, and activation timing.
- Infinite, Immeasurable, and Irrelevant speed need exact mechanics. Do not treat labels as interchangeable.
- Travel speed is not always combat speed.
- Reaction speed can stop a blitz only if the profile supports it.

## Common checks

- Is the speed key from the same version as the AP key?
- Does the character start in that form?
- Is the ability passive or does it need activation?
- Does the opponent resist the faster side's win condition?
