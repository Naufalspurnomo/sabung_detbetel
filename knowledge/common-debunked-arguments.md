# Common Arguments and Rebuttals

## "Saitama has infinite power / no limiter"

- Rebuttal: no limiter is best treated as growth potential unless the current profile tier already quantifies infinite output.
- Judge action: compare current AP tier first, then discuss whether he has enough time to grow.
- Source reference: https://vsbattles.fandom.com/wiki/Saitama

## "Character X is omnipotent"

- Rebuttal: battleboarding needs demonstrated scope and limits.
- Judge action: ask for the applicable profile tier, key, and feats.
- Source reference: https://vsbattles.fandom.com/wiki/Tiering_System

## "Speed equalized means speed does not matter"

- Rebuttal: equalized speed changes movement/reaction comparisons, but abilities with travel time, activation, or range can still matter.
- Judge action: state whether the room uses speed equalized rules.

## "A stronger verse means every character scales"

- Rebuttal: verse-wide scaling is invalid unless each character has a direct chain.
- Judge action: require the chain and reject unsupported verse-scaling jumps.
